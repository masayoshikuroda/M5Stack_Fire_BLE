# M5Stack_Fire_SensorTag

M5Stack Fireの CC2650　エミュレーション.
スマホのSensorTagアプリからアクセスでき、以下のサービスを提供する

- Device Information
  - Serial(MAC Address)
- Battery Service
   - Baettery Level
- Movement Sensor(MPU9250)
  - Accelerometer
  - Gyroscope
  - Magnetometer
- SimpleKeys Service
  - Red LED(NeoPixel Left side)
  - Blue LED(NeoPixel Right side)
  - Buzzer(Speaker)
- IO Service
  - Left key(BtnA)
  - Right key(BtnB)
  - Reed Relay(BtnC)
